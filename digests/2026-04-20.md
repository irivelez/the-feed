# Daily Intelligence Digest — 2026-04-19

## Big Picture
The developer is becoming a manager: as AI coding tools get massively more capable (Codex with browser, memory, plugins; Opus 4.7 with adaptive thinking), the evidence shows humans shifting from writing code to reviewing it — and using MORE AI, not less, as it improves.

## Key Takeaways
- Codex got a massive upgrade — browser control, computer use, persistent memory, 90+ plugins, and image generation turn it into a full agentic coding environment, not just an autocomplete.
- Opus 4.7 requires real workflow changes: adaptive thinking replaces fixed budgets, version mismatches kill quality, and Boris Cherny says verification loops are the #1 lever for output quality.
- The Cursor/UChicago study is the first rigorous evidence that better AI models create a Jevons paradox — developers use MORE AI, shift to reviewing over writing, and only tackle harder problems after a volume-first phase.
- OpenAI's GPT-Rosalind marks the start of domain-specific frontier models, while Anthropic's platform play (per Mostaque) is about owning the workflow wrapper, not just the model layer.
- Google shipped multilingual TTS in Gemini 3.1 Flash, and the LangGraph/LangSmith ecosystem keeps maturing with Cisco reference architectures and reusable evaluator templates — enterprise multi-agent infrastructure is solidifying fast.
